# AloContact - Call Center BI Project

Proyecto de inteligencia de negocios para análisis de operaciones de call center.

## Descripción

Sistema de reportes y análisis para monitoreo de:
- Performance de consultoras
- Métricas de llamadas
- Análisis de campañas
- KPIs operacionales

## Tecnologías

- SQL Server
- Power BI
- Stored Procedures
- Databricks (migración)

## Estructura

- `/SQL-Queries/` - Consultas SQL principales
- `/Stored-Procedures/` - Procedimientos almacenados
- `/Documentation/` - Documentación técnica
